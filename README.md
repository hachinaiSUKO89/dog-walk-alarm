# 🐶 お犬様散歩アラーム

気象庁(JMA)の天気予報データを使って、全国の犬の散歩に適した気温かどうかを
地図・グラフでチェックできるWebアプリです。地面がアスファルトか土・芝かで
路面温度の推定値が変わり、それに応じたアドバイスを表示します。

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

起動すると `http://localhost:8501` がブラウザで開きます。

## スマホからアクセスする方法

PCとスマホが同じWi-Fiに接続されている状態で、PC側を以下のように起動します。

```bash
streamlit run app.py --server.address 0.0.0.0
```

起動時にターミナルに表示される `Network URL`(例: `http://192.168.x.x:8501`)を
スマホのブラウザで開くとアクセスできます。

外出先など別ネットワークからアクセスしたい場合は、Streamlit Community Cloud
などのホスティングサービスにデプロイしてください。

## 機能

- 全国47都道府県の天気・気温を気象庁のデータから取得(10分キャッシュ)
- 日本地図上に天気アイコンと気温を表示(Folium)
- 都道府県別の最高気温グラフ(判定レベルで色分け)
- 気温 + 地面タイプ(アスファルト/土・芝)から路面温度を概算し、
  散歩の適性を「適温 / 注意 / 危険 / 防寒注意」の4段階でルールベース判定

## 注意事項

- 路面温度の推定値は、気温との関係としてよく知られる目安値を基準にした
  概算(線形補間)であり、実測値ではありません。日射・風・時間帯・路面の
  状態によって実際の値は大きく変動します。真夏の炎天下では、可能であれば
  実際に手のひらで路面温度を確認してから散歩することをおすすめします。
- 気象庁の非公式JSON APIを利用しています。仕様変更により取得できなくなる
  可能性があります。
